Propiedades básicas de CSS, Display: inline, block, inline-block



Objetivos



Objetivo General


Desarrollar sitios web en formato HTML5 y CSS3 por medio del uso de diferentes editores de código


Objetivos Especificos


  • Aprender las propiedades de los CSS y los valores para darle los estilos a la página web
  • Conocer el uso de la propiedad display para tener una visualización interna y externa de un elemento.
  • Comprender y diferenciar el comportamiento entre display: inline, display:inline-block y display:block
